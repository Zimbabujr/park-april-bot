# 🚗 Бот карпулинга «Парк Апрель»

Бот для организации совместных поездок жителей района Парк Апрель.

## 📍 Информация о районе
- **Название:** Парк Апрель
- **Координаты:** 55.529598, 37.032715 (Яндекс Карты)
- **Радиус:** 2 км

## 🚀 Установка

1. Клонируйте репозиторий
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate  # Windows
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Создайте файл `.env` на основе `.env.example`
5. Запустите бота:
   ```bash
   python main.py
   ```

## 🔧 Настройка

### Создание бота
1. Напишите @BotFather, создайте бота, получите токен
2. Добавьте бота в общий чат района и сделайте администратором
3. Создайте канал для публикаций, добавьте бота администратором

### Получение ID чатов
- Отправьте любое сообщение в чат/канал
- Перейдите по ссылке: `https://api.telegram.org/bot<TOKEN>/getUpdates`
- Найдите `"chat":{"id":-100...`

## 📋 Функционал

| Функция | Описание |
|---------|----------|
| Регистрация | ФИО, адрес, телефон, данные авто |
| Создание поездки | Маршрут, время, места, цена |
| Поиск | По дате, направлению |
| Бронирование | Заявка → подтверждение |
| Жалобы | На пользователя с указанием причины |
| Блокировка | Автоматическое исключение из чата |
